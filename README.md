# Selenium Automation Framework

## Tech Stack
- Java 21
- Selenium WebDriver
- TestNG
- Maven
- Apache POI
- Extent Reports
- Log4j
- Jenkins

## Features
- Page Object Model (POM)
- Cross Browser Testing
- Data Driven Testing
- ThreadLocal WebDriver
- Screenshot Capture
- Retry Analyzer
- Jenkins Pipeline Integration

## Execute Tests

mvn clean test
